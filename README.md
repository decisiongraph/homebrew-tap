# Homebrew Tap for Decision Graph

## Installation

```bash
brew install decisiongraph/tap/dg
```

Or:

```bash
brew tap decisiongraph/tap
brew install dg
```

## About

[Decision Graph](https://github.com/decisiongraph/dg) is a text-based knowledge graph for company decisions. Capture decisions, strategies, and institutional knowledge in markdown files that version control alongside your codebase.
